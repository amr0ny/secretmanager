# secretmanager

CLI-утилита для генерации и сохранения секретов.

## Установка

```bash
go install github.com/alexalexdidit/secretmanager@latest
```

Если нужен вариант через директорию `cmd`:

```bash
go install github.com/alexalexdidit/secretmanager/cmd/secretmanager@latest
```

## Использование

```bash
secretmanager generate --key-size 32
```
